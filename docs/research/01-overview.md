# Обзор методов 3D-реконструкции лица

## Введение

3D-реконструкция лица по фотографиям — задача компьютерного зрения, направленная на восстановление трёхмерной геометрии и текстуры лица из двухмерных изображений. Данная область активно развивается с 1990-х годов и охватывает широкий спектр методов — от классической фотограмметрии до современных нейросетевых подходов.

## Классификация методов

### По количеству входных изображений

**Монокулярные методы** (1 фото):
- 3D Morphable Models (3DMM)
- Deep Learning based Reconstruction (DECA, EMOCA, MICA)
- Single-image depth estimation

**Малоизмерные методы** (2–10 фото):
- Stereo reconstruction
- Structure from Motion (SfM)
- Multi-view stereo (MVS)

**Многоизмерные методы** (10+ фото):
- Фотограмметрия
- Dense stereo reconstruction
- Light stage scanning

### По подходу к представлению

**Параметрические модели:**
- 3DMM (Basel Face Model, SCAPE, FLAME)
- Вход: вектор параметров (shape, expression, pose)
- Выход: 3D-меш с фиксированной топологией

**Непараметрические модели:**
- Point clouds
- Volumetric representations
- Neural Radiance Fields (NeRF)

**Гибридные подходы:**
- DECA: 3DMM + детальная карта смещений
- MICA: 3DMM + метрическая калибровка

### По методу восстановления

**Оптимизация на основе физики:**
- Photometric stereo
- Shape from Shading
- Structured light scanning

**Обучение на данных (Data-driven):**
- Supervised learning
- Self-supervised learning
- Generative models (GAN, VAE)

**Комбинированные подходы:**
- Analysis-by-synthesis
- Differentiable rendering

## Ключевые метрики качества

| Метрика | Описание | Типичные значения |
|---------|----------|-------------------|
| Chamfer Distance | Расстояние между облаками точек | 0.5–2.0 мм |
| Normal Consistency | Согласованность нормалей поверхностей | 0.85–0.95 |
| F-Score | Точность и полнота восстановления | 0.7–0.9 |
| PSNR | Качество реконструкции текстуры | 25–35 дБ |
| SSIM | Структурное сходство | 0.8–0.95 |

## Проблемные области

### Domain Gap

Разрыв между тренировочными и реальными данными остаётся ключевой проблемой. Модели, обученные на контролируемых датасетах (BIWI, MEAD), могут poorly generaliz на произвольных фотографиях.

### Освещение и экспозиция

Изменения освещения significantly влияют на качество реконструкции. Тени, блики и пересвет искажают восприятие формы.

### Выражения лица (Expressions)

Динамические изменения мимики (улыбка, нахмуренные брови) требуют дополнительных параметров для точного моделирования.

### Экстремальные углы

Реконструкция при extreme angles (profile, upside-down) остаётся сложной задачей для большинства методов.

## Выводы

Современные методы 3D-реконструкции лица достигли высокого качества при minimal входных данных. Наиболее перспективным направлением для practical применения являются deep learning подходы (DECA, EMOCA, MICA), обеспечивающие баланс между качеством, скоростью и minimal требованиями к оборудованию.
