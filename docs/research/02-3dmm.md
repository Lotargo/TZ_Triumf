# 3D Morphable Models (3DMM)

## Введение

3D Morphable Models (3DMM) — параметрические модели, описывающие трёхмерную геометрию лица в виде линейной комбинации базисных векторов (principal components). Модель позволяет представить任意的脸部形状 и expression небольшим числом параметров, что делает её удобной для анализа, реконструкции и генерации.

## Математический аппарат

### Основное уравнение

Форма лица S описывается как:

```
S = S̄ + Σ αᵢ · sᵢ
```

Где:
- S̄ — средняя форма (mean face)
- αᵢ — вес i-го principal component
- sᵢ — i-й principal component (basis vector)

### Разложение на компоненты

Полная модель включает:

```
S = S̄ + Bₛ · α + Bₑ · β + Bₚ · γ
```

- **Bₛ** — базис формы (shape basis)
- **Bₑ** — базис выражений (expression basis)
- **Bₚ** — базис позы (pose basis)
- **α, β, γ** — соответствующие параметры

### Количество параметров

| Компонент | Типичное кол-во | Описание |
|-----------|-----------------|----------|
| Shape | 100–300 | Индивидуальные особенности лица |
| Expression | 50–100 | Динамика мимики |
| Pose | 6–9 | Углы поворота, положение |
| Texture | 100–300 | Карта альбедо |

## Ключевые модели

### Basel Face Model (BFM)

Разработан в University of Basel. Одна из наиболее широко используемых 3DMM.

**Характеристики:**
- Версия 2009: 199 shape components, 199 texture components
- Версия 2017: 199 shape + 100 expression components
- Топология: 53,490 вершин, 106,930 треугольников
- Датасет: 200 сканированных лиц

**Преимущества:**
- Хорошая generalization
- Стандарт де-факто для исследований
- Открытая лицензия для исследований

**Ограничения:**
- Ограниченная детализация (net фиолетовых details)
- Фиксированная топология
- Не учитывает детали формы (wrinkles, pores)

### SCAPE (Shape Completion and Animation of People)

Модель для полного тела, но адаптированная для лица.

**Характеристики:**
- Раздельное моделирование формы и expression
- Артикулируемая модель (articulated model)
- Использует blend shapes для анимации

### FLAME (Face LiDAR-Aided Generative Model)

Современная модель, разработанная в Max Planck Institute.

**Характеристики:**
- 4,000 вершин (оптимизировано для real-time)
- Shape basis: 100 компонентов
- Expression basis: 50 компонентов
- Neck, jaw, и eye parameters
- Открытая лицензия для исследований

**Преимущества:**
- Лёгкая и быстрая
- Хорошая детализация
- Поддержка анимации

**Ограничения:**
- Ограниченная детализация текстуры
- Требует дополнительного model для текстуры

## Методы реконструкции 3DMM

### Analysis-by-Synthesis

Классический подход, основанный на итеративной оптимизации:

```python
# Псевдокод Analysis-by-Synthesis
for iteration in range(max_iter):
    # 1. Рендер текущей модели
    rendered = render_3dmm(params)
    
    # 2. Вычисление ошибки
    loss = compute_loss(rendered, target_image)
    
    # 3. Обновление параметров
    params = params - learning_rate * gradient(loss, params)
```

**Преимущества:**
- Высокая точность при хорошей инициализации
- Физически обоснованная модель

**Ограничения:**
- Вычислительно дорого
- Зависимость от начальной инициализации
- Проблема local minima

### Regression-based Methods

Нейросетевые модели, предсказывающие параметры 3DMM напрямую из изображения:

```python
# Архитектура регрессора
class FaceReconstructor(nn.Module):
    def __init__(self):
        self.encoder = ResNet50(pretrained=True)
        self.shape_head = nn.Linear(2048, 199)  # Shape params
        self.expr_head = nn.Linear(2048, 100)   # Expression params
        self.pose_head = nn.Linear(2048, 6)     # Pose params
    
    def forward(self, x):
        features = self.encoder(x)
        shape = self.shape_head(features)
        expr = self.expr_head(features)
        pose = self.pose_head(features)
        return shape, expr, pose
```

**Преимущества:**
- Быстрый инференс (10–50 мс)
- End-to-end обучение
- Хорошая generalization

**Ограничения:**
- Зависимость от качества данных
- Ограниченная детализация

### Differentiable Rendering

Современный подход, объединяющий анализ и синтез:

```python
# Differentiable rendering pipeline
class DifferentiableRenderer:
    def forward(self, params):
        # Рендер с градиентами
        rendered, gradients = self.renderer(params)
        return rendered, gradients
    
def train_step(model, image, renderer):
    params = model(image)
    rendered, grads = renderer(params)
    loss = F.l1_loss(rendered, image)
    loss.backward()  # Градиенты через рендерер
    return params
```

**Преимущества:**
- Объединяет преимущества обоих подходов
- End-to-end дифференцируемый
- Высокая точность

**Ограничения:**
- Вычислительно дорого
- Требует GPU

## Ограничения 3DMM

### Фиксированная топология

Все лица имеют одинаковое количество вершин и треугольников. Это ограничивает:
- Детализацию (net фиолетовых details)
- Адаптацию к уникальным особенностям
- Восстановление аксессуаров (очки, piercing)

### Линейная модель

3DMM предполагает линейную комбинацию базисных векторов. Это:
- Ограничивает表达能力
- Не моделирует nonlinear interactions
- Может приводить к артефактам

### Отсутствие текстуры

Базовая 3DMM не включает текстуру. Требуются дополнительные методы:
- Texture-from-vertex coloring
- Neural texture synthesis
- UV-mapping с проекцией

## Современные developments

### Neural 3DMM

Замена линейных базисных векторов на нейросетевые представления:

```python
# Neural 3DMM
class Neural3DMM(nn.Module):
    def __init__(self):
        self.decoder = nn.Sequential(
            nn.Linear(latent_dim, 256),
            nn.ReLU(),
            nn.Linear(256, 512),
            nn.ReLU(),
            nn.Linear(512, num_vertices * 3)
        )
    
    def forward(self, latent):
        vertices = self.decoder(latent)
        return vertices.view(-1, 3)
```

### 3DMM + Deep Learning

Гибридные подходы (DECA, EMOCA):
- Используют 3DMM как регуляризатор
- Добавляют детальную карту смещений
- Сохраняют физическую интерпретируемость

## Выводы

3DMM остаётся фундаментальным инструментом для 3D-реконструкции лица. Несмотря на ограничения (фиксированная топология, линейность), модель обеспечивает:
- Физически обоснованное представление
- Интерпретируемые параметры
- Эффективную компрессию

Современные подходы (DECA, EMOCA) расширяют возможности 3DMM, добавляя детализацию и улучшая generalization.
